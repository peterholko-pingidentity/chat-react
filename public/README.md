# TAPGUN Logo

## Instructions

Please save the TAPGUN logo image (the one with red stripes, white text, and star) to this directory as:

```
tapgun-logo.png
```

### Logo Requirements:
- **Format**: PNG (preferred) or SVG
- **Recommended dimensions**: Width 400-600px, Height 150-200px
- **Background**: Transparent or black
- **File name**: Exactly `tapgun-logo.png`

The logo will be displayed in the header of the chat application.

If the logo file is not found, the app will display a fallback text logo "[TAPGUN]" in the military/monospace font.

---

## Quick Setup

If you have the logo image file, simply copy it here:

```bash
cp /path/to/your/tapgun-logo.png /home/user/chat-react/public/tapgun-logo.png
```

Then restart the development server to see the changes.
